Tab Descrição
Registra dados sobre as principais informações de identificação de uma doação, como número do contrato, parceiro, período de vigência. A cor dourada significa que é a tab selecionada.
IMPORTANTE: para novos cadastro somente essa tab irá aparecer, as demais só apareceram após o registro das informações gerais.
Registra dados a respeito de processos que serão permitidos para a doação e linha orçamentária.
IMPORTANTE: Só será vizualizada após o preenchimento e salvamento das informações gerais
Registra dados a respeito do planejamento da doação, divisão de orçamento por linha orçamentária e gestão.
IMPORTANTE: Só será vizualizada após o preenchimento e salvamento das informações gerais
Campo Descrição
Fonte Quem fez a doação (Bradesco, Samsung ou qualquer outro que esteja na lista). É um campo de autocompletar, ao digitar aparecerá uma lista para que seja selecionado um parceiro
OBS: É um campo de autocompletar e é necessário selecionar na lista suspensa que aparecerá de acordo com o que foi digitado no campo
Título/Identificação Uma descrição simples que identifique essa doação
Contrato Número do documento jurídico firmado entre a instituição (FAS) e nossos parceiros (Fontes)
Conta Bancária Número da conta bancária (FAS) em que deve ser usada na doação, tanto para receber os valores acordados em contrato quanto para desembolso/investimentos do projeto
Tipo de controle Determina como o sistema vai bloquear o saldo e não deixar que seja gasto acima do valor disponível.
Orçamento: Aponta que o sistema vai bloquear levando em consideração o campo 'Valor orçado' do orçamento.
Caixa: Aponta que o sistema vai bloquear levando em consideração o valor que já foi depositado na conta informada no campo 'conta', ou seja, valor em caixa.
Tipo de doação Informa se a doação é vinculada a linhas orçamentárias ou desvinculada.
Vinculada: É obrigatório a divisão por linha orçamentária.
Desvinculada: Não há necessidade de divisão por linha orçamentária deixando a critério do gestor definir como irá gastar na definição dos projetos
Data inicial Determina quando começa o período de vigência de uma doação
Data final Determina quando termina o período de vigência de uma doação
Responsável técnico Colaborador responsável por gerenciar e executar e acomapanhar os orçamentos no setor tecnico de projetos
OBS: É um campo de autocompletar e é necessário selecionar na lista suspensa que aparecerá de acordo com o que foi digitado no campo
Analista financeiro Colaborador responsável pela análise financeira e controladoria dos recursos da doação
OBS: É um campo de autocompletar e é necessário selecionar na lista suspensa que aparecerá de acordo com o que foi digitado no campo
Valor orçado Valor orçando e contratado entre a instituição (FAS) e nosso parceiro (Fonte).
Campo Descrição
Permite empréstimo? Determina se uma doação pode emprestar valores de doação para outros projetos.
Permite remanejamento entre categorias? Determina se uma doação pode remanejar valores entre categorias da mesma doação
Permite remanejamento entre linhas orçamentárias? Determina se uma doação pode remanejar valores entre Linhas orçamentárias
Paga tarifa bancária? Determina se uma doação deve ou não pagar tarifa.
OBS: Se for marcado será necessário também informar o tipo de controle do pagamento de tarifa
Tipo de controle? Determina, em relação a projeto, quem paga a tarifa bancária (Quando necessário pagar).
Mesmo projeto: Determina que o projeto selecionado na solicitação (SC,SP,SA,SD).
Outro projeto: Determina que outro projeto deverá pagar a tarifa, caso seja outro é necessário escolher o projeto e a linha orçamentária (SC,SP,SA,SD).
Projeto Determina qual projeto (outro projeto que não seja da doação) pagará a tarifa bancária.
OBS: Só é ativado caso seja escolhido a opção 'Outro projeto' no campo 'Tipo de controle'.
OBS: É um campo de autocompletar, deve ser escolhido um projeto na lista suspensa de acordo com o que foi digitado no campo
Linha orçamentária Determina qual linha orçamentária pagará a tarifa bancária.
Campo Descrição
Incluir nova linha
Direciona para um formulário onde permite cadastrar novas linhas orçamentárias
Divisão de orçamento
Direciona para um formulário onde permite cadastrar valores por gestão, dividindo o valor por setor da instituição e atribuindo um gestor responsável
Componente Parte de uma doação que determina um echo de atuação.
Subcomponente Parte de uma componente que determina uma atividade do componente.
Categoria Dotação orçamentária ou rubrica de gastos de uma doação.
Valor disponível Valor disponível na linha orçamentária para liberar em projetos.
Valor empenhado Valor projetizado ou distribuído para projetos referente linha orçamentária.
Valor Valor orçado para linha orçamentária (componente, subcomponente e categoria).
