Campo Descrição
Fonte Quem fez a doação (Bradesco, Samsung ou qualquer outro que esteja na lista). É um campo de autocompletar, ao digitar aparecerá uma lista para que seja selecionado um parceiro
Título Uma descrição simples que identifique essa doação
Data inicial Determina quando começa o período de vigência de uma doação
Data final Determina quando termina o período de vigência de uma doação
Responsável técnico Colaborador responsável por gerenciar e executar e acomapanhar os orçamentos no setor tecnico de projetos
Analista financeiro Colaborador responsável pela análise financeira e controladoria dos recursos da doação
Botões Descrição
Buscar
Filtra registros de acordo com os campos preenchidos nos campos de filtros
Novo cadastro
Direciona para o formulário de cadastro de doações
Direciona para o formulário de cadastro de doações com as informações do registro selecionado
Remove uma doação, só funcionará caso a doação não tem associações com linhas orçamentárias ou com projetos
Ordena a lista de doações, é possível ordenar por data início, data final e valor. A ordem pode ser crescente ou decrescente de acordo com a seta na coluna
Paginação da tabel para passar para próximas páginas de registros
Ajuda
Tutorial para ajudar a entender página
